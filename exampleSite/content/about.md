The Clean Hugo theme is a modern, customizable Hugo theme built with Vanilla CSS and Alpine.js.

## Features

- **Responsive Design**: Looks great on all devices
- **Customizable Colors**: Easy theme customization
- **Package & People Listings**: Built-in package showcase functionality driven by yml data files and Alpine.js.
- **Blog Support**: ((Full-featured--*coming soon*)) blog with categories and tags
- **Gallery Views**: Beautiful responsive image galleries that use masonry layout.
- **Shortcodes**: Powerful content components for cards, metrics, sections, and more.

## Built With

- Hugo static site generator
- Vanilla CSS for styling
- AlpineJS for interactivity
- Font Awesome icons
