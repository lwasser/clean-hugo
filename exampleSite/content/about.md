# About Clean Hugo Theme

The Clean Hugo theme is a modern, customizable Hugo theme built with Tailwind CSS.

## Features

- **Responsive Design**: Looks great on all devices
- **Customizable Colors**: Easy theme customization
- **Package Listings**: Built-in package showcase functionality
- **Blog Support**: Full-featured blog with categories and tags
- **Gallery Views**: Beautiful image galleries
- **Shortcodes**: Powerful content components

## Built With

- Hugo static site generator
- Tailwind CSS for styling
- AlpineJS for interactivity
- Font Awesome icons
